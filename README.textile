h1. rah_beacon

Create dynamic alias templates tags from Textpattern CMS' form partials.

h2. Discontinued

This is now part of "Textpattern CMS":https://textpattern.com core as of version 4.7.0. Templates can now be called using short tags without any extra plugins. For more information, see "here":https://docs.textpattern.com/tags/output_form.

This plugin, and repository, never had a stable release, so this is no effectively discontinued. This repository is kept around so that existing installs that may be using the Composer package do not break and for history reasons.

h2. Install

Using "Composer":https://getcomposer.org:

bc. $ composer require rah/rah_beacon:*@dev
